//bash description
Bourne Again SHell.
Имя разработчика: Чет Рамей. 
Поддерживает богатый язык написания скриптов, удобный интерфейс для редактирования командной строки, автопродолжение команд и множество других полезных возможностей.
//sh description
Bourne Shell.
Имя разработчика: Стивен Борн. 
sh поддерживает условные операторы (if, case), циклы (for, while), функции и переменные, необходимые для создания скриптов.
//tcsh description
Tenex C Shell.
Имя разарботчика: Кэн Грир. 
Главной особенностью оболочки стала возможность автодополнения путей и команд. Также tcsh позвоялет проверять правописания имён файлов, команд и переменных.
//zsh description
Z shell.
Имя разработчика: Пауль Фалстад. 
Поддерживает все команды Bash, а также свои уникальные команды и параметры конфигурации, например, autoload, zle.
